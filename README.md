# Only

## Скрипты

| Скрипт          | Назначение                                  |
| --------------- | ------------------------------------------- |
| `npm run dev`   | Запуск dev-сервера на http://localhost:5000 |
| `npm run build` | Сборка проекта в `dist/`                    |

## Установка

```bash
git clone https://github.com/KirillBarabanshikov/only.git
cd only
npm install
```
